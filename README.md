# What is this?

Talk Balloons is a client side mod that allows you to see bubbles above the players heads when they send messages to the chat.

The bubbles adjust to the size of the message, with larger messages being supported too.

Through the config file you can change stuffs like: the max/min balloon width, the text color, the amount of balloons rendered and more.

This mod requires [KotlinLangForge](https://modrinth.com/mod/kotlin-lang-forge).

### QSMP Live Translation:

This mod is now maintained by [BluSpring](https://bluspring.xyz), who has created an addon for this mod that replicates the QSMP's live translation system, which allows for transforming speech into text, and making them translatable to other languages. You may find it under [UnityTranslate](https://modrinth.com/mod/unitytranslate).
